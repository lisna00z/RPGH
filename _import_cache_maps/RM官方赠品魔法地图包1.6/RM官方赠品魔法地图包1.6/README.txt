■Warning
RPG Maker MV Ver 1.6.1(Incompatible with versions below 1.6.1)
This plugin is made for RPG Maker MV version 1.6.1.
Please refer to the sample project for proper usage.

■About the effect plug-in

Place the plugins in the following order in the PluginManager.

SAN_ShaderFilter.js
SAN_ShaderFilterCharaRipple.js
SAN_ShaderFilterCharaHeatHaze.js
SAN_ShaderFilterScanLine.js

To make use of these effects, some scripting will be required.
Please refer to the sample project for details.
